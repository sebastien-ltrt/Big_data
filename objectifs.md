Objectif du projet 
 

Concevoir et mettre en œuvre un pipeline de données complet allant de la collecte à la visualisation, en combinant des données issues d’une API et du scraping web.

Le projet couvre :

Ingestion de données depuis une API .
Scraping de données .
Stockage des données brutes (Data Lake local)
Transformation des données avec Python
Chargement dans PostgreSQL (Data Warehouse)
Création d’un tableau de bord interactif
Préparation d’une présentation de soutenance .
 
Technologies utilisées
Python
API (requests)
Scraping (BeautifulSoup / Selenium)
Transformation (pandas)
Data Lake (local)
JSON / CSV
PostgreSQL
Stockage structuré
Dashboard
Streamlit
ou Power BI
 

Workflow du projet
Ingestion (API + scraping)
Stockage brut
Transformation et fusion
Chargement en base
Visualisation
